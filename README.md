# Hotel Management (Grand Hotel)

Laravel 12 hotel management app with:
- Website pages (rooms, services, gallery, contact, etc.)
- User auth + booking + dummy payment + invoice PDF
- Admin panel + CMS content management

## Quick Start

```bash
copy .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan storage:link
php artisan serve
```

Admin login (seeded):
- `admin@hotelier.com` / `password`
